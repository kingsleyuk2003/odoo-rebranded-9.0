Developer tweaks

... to make a developer's life easier...

When you're coding you keep repeating some operations over and over (like updating modules). And after a day's work you've spent quite some time jumping from one screen to the other or typing prompt commands. This is boring and mind consuming.

"Developer tweaks" brings you some features that will help you while developing Odoo modules.

All you have to do is install this module and, then, associate your user to the "Odoo Developer" group.

Easier access to modules

1 click update

Instead of having to go through three screens in order to update a module, you just have to push the top menu link that shows the latest module to be updated/installed.

Latest updates

Modules deliberately updated/installed (that means you actually pushed a button) are listed so that you know what was done in the last couple of days.

Always a way

Wherever there is a reference to a module, there is a button (or an icon) to install/update it.
The less you click, the less time you waste.

Know where you are

It's easy to log in into the wrong database when you have a bunch of them in your server.

Now, the database's name shows on top of the company's logo, so that you'll always know where you are.
